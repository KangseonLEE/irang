## 8. Kill Criteria 대응 전략

> 4/17 기준 10 UV/day, 5/3 기준 50 WAU

### UV 확보 (즉시~1주)

1. **SEO 사이트맵 크롤링 정상화** → 구글 자연 유입 시작
2. **네이버 서치어드바이저 등록** → 국내 검색 유입 확보
3. **JSON-LD 구조화 데이터** → 검색 결과 리치 스니펫 노출
4. **OG 이미지 품질** → SNS 공유 시 클릭률 향상

### WAU 확보 (2~4주)

1. **진단(assess) → 매칭(match) → 지역 상세** 전환 퍼널 최적화
2. **마감 임박 알림** → 재방문 동기 부여
3. **나의 준비현황** → 진행 상태 확인 재방문
4. **피드백 수집** → 실 사용자 니즈 기반 빠른 개선
